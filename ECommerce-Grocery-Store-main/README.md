# E-Commerce Web Application (Flask + MySQL)

This is a full-stack E-Commerce web application developed as part of an internship project.

## Tech Stack
- HTML5
- CSS3
- Bootstrap
- JavaScript
- jQuery
- Python (Flask)
- MySQL

## Features
- User Login & Registration
- Product Listing
- Cart Management
- Admin Panel
- Database Integration
- Secure Backend using Flask

## Setup Instructions
1. Clone the repository
2. Create virtual environment
3. Install dependencies
4. Configure database.yaml
5. Run the project using `python run.py`
